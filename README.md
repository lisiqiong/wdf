# wdf
***
目的
- 1、为了深入学习mvc思想
- 2、更深入的了解面向对象的设计模式以及实际应用
- 3、了解框架的运行原理

框架名称命名：框架名称wdf为 Website Development Framework（网站开发框架）的缩写

一、单入口
定义：进入框架只允许有唯一的入口文件且有严格的生命周期
### 优点：
- （1）安全性高
- （2）可读性强
- （3）更规范，方便管理

### 实现：
- （1）：强实现，使用域名重写urlRewrite模块来实现
- （2）：软实现，入口文件我们使用面向过程的写法，框架其它的都采用面向对象的写法，要访问类中的方法，必须通过入口文件的调配。
