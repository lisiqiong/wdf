## 一、单入口
***
定义：进入框架只允许有唯一的入口文件且有严格的生命周期
### 优点：
- 1、安全性高
- 2、可读性强
- 3、更规范，方便管理

### 实现：
- 1、强实现，使用域名重写urlRewrite模块来实现
- 2、软实现，入口文件我们使用面向过程的写法，框架其它的都采用面向对象的写法，要访问类中的方法，必须通过入口文件的调配。
